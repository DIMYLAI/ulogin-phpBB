Форма авторизации uLogin через социальные сети. Улучшенный аналог loginza.

== Description ==

uLogin — это инструмент, который позволяет пользователям получить единый доступ к различным Интернет-сервисам без необходимости повторной регистрации,
а владельцам сайтов — получить дополнительный приток клиентов из социальных сетей и популярных порталов (Google, Яндекс, Mail.ru, ВКонтакте, Facebook и др.)

== Installation ==

Откройте файл install_mod.xml в вашем браузере и следуйте инструкциям.

Замечания:

1. При чтении руководства файлы install_mod.xml и modx.prosilver.en.xsl должны находиться в одной директории
2. При создании таблицы в базе данных может возникнуть необходимость изменить префикс создаваемой таблицы (phpbb_) на тот, что используется у вас
3. После редактирования шаблонов не забудьте сбросить кэш шаблонов: Стили -> Шаблоны. Нажать кнопку Обновить у текущего шаблона.
